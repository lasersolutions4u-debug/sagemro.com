## 二、各渠道详细指南

### 2.1 知乎

**定位：** 行业专家人设，专业深度内容

**打法：**
- 搜索"激光切割""折弯机""焊接参数"等话题，逐条回答
- 回答结构：问题分析 → 原因 1/2/3 → 解决方案 → 安全提示
- 回答末尾自然带出："更多设备问题可以来 SAGEMRO 找小智聊聊"
- 每周 3-5 条回答 + 每月 1 篇深度专栏文章

**禁忌：**
- 不要硬广。先给价值，再软引导
- 不要复制粘贴同一段话到多个问题
- 不要在回答里放二维码（容易被折叠）

### 2.2 微信公众号

**定位：** 用户留存池，品牌信息中枢

**内容规划：**
- 周一：设备维护小贴士（实用短篇）
- 周三：故障排查案例（从工单脱敏转化）
- 周五：行业资讯/设备选购指南（可选）

**运营要点：**
- 菜单栏设"问小智"入口，直接跳转 sagemro.com
- 新关注自动回复：引导用户描述设备问题
- 文末固定 CTA："设备有故障？让 AI 小智帮你诊断 →"

### 2.3 抖音

**定位：** 广泛曝光，触达一线操作工和维修师傅

**内容方向：**
- 设备故障实拍 + 诊断方案（核心）
- 维修前后对比（爽感）
- 设备保养小技巧（实用）
- 行业冷知识/设备冷知识（趣味）

**拍摄要点：**
- 前 3 秒必须有钩子（故障画面/问题喊话/惊人对比）
- 竖屏 9:16，字幕清晰
- 背景音乐音量低于人声
- 使用设备维修相关热门话题标签

**脚本模板：**
```
0-3s:  设备故障特写 + "激光切割全是毛刺？别慌"
3-15s: 快速说 2-3 个可能原因（配合画面）
15-25s: 展示关键调整动作或参数
25-30s: "更多问题来 SAGEMRO 找小智"
```

### 2.4 视频号

**定位：** 微信生态内短视频，偏专业实用

与抖音内容可复用，但：
- 时长可稍长（30-60s）
- 语速可稍慢，更强调专业感
- 在视频描述中挂公众号链接或官网链接

### 2.5 小红书

**定位：** 触达年轻工程师和管理者

**内容方向：**
- 设备保养清单（图文条漫风格）
- 工厂日常（建立亲近感）
- 设备选购指南（对比表格）
- 维修师傅的一天（人物故事）

**发布要点：**
- 封面图要精致，标题要勾人
- 多用清单体、对比体
- 话题标签用：`#钣金加工` `#激光切割` `#工厂日常` `#设备维修`

### 2.6 百家号

**定位：** 百度生态内容分发，辅助 SEO

- 官网博客文章同步发布（注意间隔 1 天，避免百度判重复）
- 文末可以放官网链接
- 百家号内容会被百度优先索引

## 三、内容复用矩阵

一份内容，多渠道变形：

```
官网技术文章（1500字）
  ├── → 知乎专栏原文发布
  ├── → 公众号精简版（800字）
  ├── → 百家号同步（间隔1天）
  ├── → 抖音脚本（提炼 1 个核心点，30s）
  ├── → 视频号脚本（提炼 2-3 个要点，60s）
  ├── → 小红书图文（清单化呈现）
  └── → 推特/微博（提炼金句，1-2 条）
```
