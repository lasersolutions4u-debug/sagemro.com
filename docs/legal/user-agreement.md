# SageMRO 用户服务协议

**最后更新日期：2026年4月24日**

**生效日期：2026年4月24日**

欢迎您使用 SageMRO 平台（以下简称"本平台"）。本协议是您与济南钰峭机械有限公司（以下简称"我们"）之间关于使用本平台服务所达成的协议。请您在使用本平台前仔细阅读本协议全部内容。

## 一、服务说明

1.1 本平台是面向钣金加工行业设备后服务市场的信息技术服务平台，提供 AI 技术咨询、设备维修工单管理、工程师匹配及交易撮合等信息技术服务。

1.2 本平台的 AI 助手"小智"提供的技术咨询内容仅供参考，不构成专业维修建议或维修承诺。具体维修方案应以现场专业人员的实际判断为准。

1.3 本平台作为信息技术服务提供方，为客户与工程师（维修服务提供方）提供信息撮合与交易管理服务。本平台本身不直接提供设备维修服务，不是维修服务的提供方或承包方。

1.4 本平台在交易过程中提供代收代付服务：受客户委托代为收取维修服务费用，并在扣除平台技术服务费后将维修服务费转付给工程师。平台代收代付的维修服务费不属于平台营业收入。

## 二、用户注册与账户

2.1 您在注册时应提供真实、准确、完整的个人信息，并在信息变更时及时更新。

2.2 您应妥善保管账户信息和密码，因账户密码保管不善导致的损失由您自行承担。

2.3 一个手机号只能注册一个账户。您不得将账户转让、出借给他人使用。

2.4 我们有权对您的注册信息进行审核，如发现信息不真实、不准确，我们有权拒绝提供服务或限制账户功能。

## 三、客户权利与义务

3.1 客户可通过本平台提交设备维修工单，查看工单进度，与工程师沟通，并对服务进行评价。

3.2 客户提交工单时应如实描述设备故障情况、设备型号等信息，以便工程师准确判断和报价。

3.3 客户在确认工程师报价后，应按约定支付费用。客户支付的总费用由以下两部分组成：

- **维修服务费**：由工程师报价确定，用于支付工程师的维修服务，由平台代收后转付给工程师；
- **平台技术服务费**：由平台按工程师等级费率收取，用于平台运营、AI 技术维护、交易管理等信息技术服务。

3.4 上述两项费用在工单报价确认页面分别列明，客户确认报价即视为同意支付上述全部费用。

3.5 客户应配合工程师的现场维修工作，提供必要的工作条件和安全环境。

## 四、工程师权利与义务

4.1 工程师是在本平台注册并通过审核的独立维修服务提供方，与本平台之间不构成雇佣、劳动或代理关系。

4.2 工程师应如实填写专业背景信息，包括擅长的设备类型、品牌、维修项目和服务地区。

4.3 工程师接单后应按约定时间响应，提供专业、负责的维修服务。

4.4 工程师应合理报价。本平台的 AI 系统会对报价进行合理性审核，仅供参考。

4.5 工程师的维修服务费由平台代收后转付。平台按工程师等级收取技术服务费：初级工程师 20%、高级工程师 15%、专家工程师 12%，该费用从客户支付的总费用中扣除后，剩余部分转付给工程师。

4.6 工程师可申请提现，提现将在 T+1 个工作日内处理。

## 五、费用结构与代收代付

5.1 本平台采用代收代付模式进行交易资金管理：

- 客户支付的总费用由**维修服务费**和**平台技术服务费**两部分组成；
- 平台代客户收取全部费用后，将维修服务费转付给工程师，平台技术服务费作为平台信息技术服务收入；
- 平台技术服务费费率按工程师等级确定：初级 20%、高级 15%、专家 12%。

5.2 平台技术服务费在工程师提交报价、客户确认后自动计算，并在工单完成结算时从代收款项中扣除。

5.3 具体费用明细以工单报价确认页面显示为准，维修服务费和平台技术服务费分别列示。

5.4 发票说明：平台就其收取的技术服务费部分向客户开具信息技术服务类发票。维修服务费部分的发票由工程师（维修服务提供方）按其主体资质自行开具。

## 六、评价体系

6.1 工单完成后，客户可从时效性、技术熟练程度、沟通流畅度、专业性四个维度对工程师进行评价。

6.2 评价应基于真实的服务体验，不得恶意差评或虚假好评。

6.3 工程师的评分将影响其在平台的推荐排名和等级晋升。
