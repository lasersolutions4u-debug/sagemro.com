## 一、买家付款到工程师收款/提现的完整流程

SAGEMRO 是一个双边撮合平台，客户（买家）通过 AI 助手"小智"提交设备维修工单，平台匹配工程师（卖家）上门服务。资金流如下：

```
① 客户向 AI 小智描述故障 → AI 如需人工则引导提交工单
     ↓
② 工程师接单 → 在工单内提交报价
   （人工费 + 配件费 + 差旅费 + 其他 = 小计）
     ↓
③ 客户在工单详情「报价确认」Tab 查看报价 → 确认报价
     ↓
④ 客户点击「去付款」→ 弹出支付弹窗
   选择支付方式（银行转账 / 支付宝 / 微信）→ 确认付款
   （接入 Airwallex 后走真实支付网关）
     ↓
⑤ 付款完成 → 工单状态进入"服务中"
   工程师上门服务 → 完成后标记 → 客户评价
     ↓
⑥ 工单完成后 → 平台按合伙人等级结算佣金
   Junior 80% / Senior 85% / Expert 88% 进入工程师钱包
   平台佣金（20%/15%/12%）留在平台账户
     ↓
⑦ 工程师在工作台钱包面板 → 点击「提现」
   提交提现金额 → 平台审核后通过 Airwallex 打款到工程师银行卡（T+1）
```

**关键节点对照表：**

| 步骤 | 操作角色 | 页面路径 | 触发条件 |
|------|---------|---------|---------|
| 创建工单 | 客户 | AI 对话 → 工单弹窗 | 客户描述设备故障 |
| 工程师报价 | 工程师 | 工单详情 → 核价 Tab | 工单状态 = 处理中 |
| 确认报价 | 客户 | 工单详情 → 报价确认 Tab | 报价已提交 |
| **发起付款** | **客户** | **工单详情 → 报价确认 Tab → [去付款] 按钮** | **报价已确认** |
| 钱包入账 | 工程师 | 工程师工作台 | 工单完成 + 客户评价 |
| **提现** | **工程师** | **工作台 → 钱包面板 → [申请提现]** | **钱包余额 > 0** |

## 二、付款入口与钱包功能的现状说明

### 2.1 付款入口

付款入口按工单生命周期动态出现。必须满足以下条件：

1. 工单已创建并分配给工程师
2. 工程师已提交报价
3. 客户已确认报价

满足后，在工单详情 →「报价确认」Tab 底部出现绿色「去付款」按钮。

### 2.2 工程师钱包

工程师工作台的钱包面板展示：
- 钱包余额（累计收入，当期 ¥2,080）
- 保证金余额
- 收入/提现流水记录（3 条）
- 「申请提现」按钮

钱包余额来自已完工单的佣金结算。提现功能可用。

### 2.3 当前支付状态

前端已完整实现支付弹窗、钱包余额、提现申请等全部 UI 和交互。当前使用平台内模拟支付（不涉及真实资金）。Airwallex 账户获批后接入：

- `POST /api/payment/create-intent` → Airwallex PaymentIntent
- `POST /api/payment/webhook` → Airwallex Webhook 回调
- Airwallex 托管支付页（PCI 合规由 Airwallex 承担）

### 6.3 完整支付流程测试（新建工单）

如需体验从零开始的完整流程，按以下步骤操作：

**第 1 步 — 客户创建工单：**
1. 客户账号 13900008888 登录
2. 在聊天框输入设备故障描述，例如：
   > "通快 TRUMPF 激光切割机 3000W，切割 6mm 碳钢严重挂渣毛刺，切割断面波纹，需要工程师上门检修。山东济南。"
3. AI 小智将引导提交工单 → 确认后工单创建成功
4. 记录工单编号（如 WO-20260514-XXX）

**第 2 步 — 工程师接单并报价：**
1. 工程师账号 13900018888 登录
2. 左侧「工程师工作台」→ 在推荐工单中找到刚才创建的工单
3. 点击工单进入详情 → 切换到「核价」Tab
4. 填写报价：人工费、配件费、差旅费等 → 提交报价

**第 3 步 — 客户确认报价并付款：**
1. 切回客户账号 →「我的工单」→ 找到工单
2. 切换到「报价确认」Tab → 查看报价明细
3. 点击「确认报价」→ 报价确认后出现「去付款」按钮
4. 点击「去付款」→ 弹出支付弹窗（显示金额和支付方式）
5. 选择支付方式 → 点击「确认付款」→ 付款成功

**第 4 步 — 工程师完成服务：**
1. 工程师账号 → 工单详情 → 标记服务完成
2. 客户账号 → 对工程师进行评价（4 个维度 + 文字）
3. 评价提交后，工单自动完成，佣金自动结算到工程师钱包

**第 5 步 — 工程师查看收入并提现：**
1. 工程师工作台 → 查看钱包余额变化（新增一笔收入）
2. 钱包流水区查看最新交易记录
3. 点击「申请提现」→ 提交提现申请

- **付款入口不是固定菜单**，需要在工单生命周期中动态触发。必须先完成"工程师报价 → 客户确认"后才会出现。
- **钱包流水**在工单完成并评价后自动生成，新工单未完成前不会有新增流水记录。
- 当前支付为平台内模拟，未接入真实支付网关。Airwallex 接入后将替换为 PaymentIntent + Webhook 模式。
