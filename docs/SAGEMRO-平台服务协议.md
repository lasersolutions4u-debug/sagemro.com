## 第一条 平台定义与服务范围

**1.1 平台定位**

SAGEMRO 是一个钣金加工设备售后服务的撮合平台。平台通过 AI 助手"小智"为设备使用者（客户）提供技术咨询，并在需要时将客户与设备维修服务提供者（工程师/合伙人）进行匹配。

**1.2 平台角色**

平台作为信息撮合方，本身不直接提供设备维修服务。维修服务由入驻平台的工程师（以下简称"工程师"）独立提供。平台对服务过程进行记录和监督。

**1.3 平台服务内容**

平台目前提供的服务包括：
- AI 技术咨询（通过"小智"对话）
- 工单创建与管理
- 工程师匹配与推荐
- 报价提交与确认
- 费用支付（当前为模拟环境，详见第六条）
- 服务评价
- 工程师钱包与提现申请

## 第三条 工单服务流程

**3.1 工单创建**

客户可通过以下方式创建工单：
- 在 AI 对话中向"小智"描述设备问题，由 AI 引导创建
- 通过侧边栏"提交工单"入口自主创建

工单创建时需填写：
- 问题类型（设备故障 / 维修保养 / 参数调试 / 其他）
- 问题描述
- 紧急程度（普通 / 紧急 / 非常紧急）

**3.2 工单状态**

工单的状态流转如下：

```
已提交(pending) → 处理中(in_progress) → 报价中(pricing) → 待付款(pending_payment) → 服务中(in_service) → 已解决(resolved) → 已完成(completed)
```

各状态说明：
- **已提交**：工单已创建，等待工程师接单
- **处理中**：工程师已接单，正在了解问题
- **报价中**：工程师已提交报价，等待客户确认
- **待付款**：客户已确认报价，等待付款
- **服务中**：客户已完成付款，工程师上门服务
- **已解决**：工程师已标记服务完成
- **已完成**：客户已提交评价，工单完结

**3.3 工单消息**

客户与工程师可在工单内进行消息沟通。消息记录将长期保存，作为服务过程的凭证。

## 第四条 报价与费用

**4.1 工程师报价**

工程师接单后可在工单内提交报价，报价包含以下项目：
- 人工费
- 配件费
- 差旅费
- 其他费用

以上四项合计为服务费小计。

**4.2 平台技术服务费**

客户支付的费用由平台统一收取，平台按服务费小计的一定比例收取平台技术服务费，剩余部分归工程师所有：

```
客户应付总额 = 服务费小计
平台技术服务费 = 服务费小计 × 平台服务费率
工程师实得金额 = 服务费小计 - 平台技术服务费
```

平台服务费率根据工程师等级确定，具体以平台公布的等级规则为准。

**4.3 报价确认**

工程师提交报价后，客户可查看完整的费用明细，选择：
- **确认报价**：接受报价，进入付款环节
- **拒绝报价**：发起议价，工程师可重新提交报价

## 第五条 工程师等级与权益

**5.1 初始等级**

工程师注册后默认为 Junior 级，信用分 100 分。等级及对应费率以平台公布的等级规则为准。

**5.2 等级信息**

工程师可在工作台查看以下信息：
- 当前等级及对应服务费率
- 信用分
- 钱包余额
- 收入流水记录

## 第六条 支付

**6.1 当前状态说明**

平台的支付功能目前处于**模拟环境**阶段。当前展示的支付方式（银行转账/支付宝/微信支付）为前端界面展示，交易为模拟交易，不涉及真实资金流动。

**6.2 正式支付上线后**

平台计划接入第三方支付网关（如 Airwallex），实现：
- 客户在线支付服务费用至平台托管账户
- 服务完成且客户确认后，平台将工程师应得款项结算至工程师钱包

正式支付上线前，平台将通知所有用户。

**6.3 托管原则**

正式支付上线后，客户支付的款项将由平台托管，在服务完成且客户确认前，款项不会直接支付给工程师。

## 第七条 工程师钱包与提现

**7.1 钱包余额**

工程师完成工单并获得客户评价后，应得款项将自动计入工程师钱包余额。工程师可在工作台查看钱包余额和收入流水。

**7.2 提现申请**

工程师可申请将钱包余额提现，但需满足以下条件：
- 已绑定银行卡信息（开户行、卡号）
- 单次提现金额不低于 100 元，不高于 50,000 元

提现申请提交后，由平台进行审核处理。若提现申请被拒绝，余额将退回钱包。

## 第八条 服务评价

**8.1 客户评价工程师**

工单完成后，客户可对工程师的服务进行评价，评价维度包括：
- 时效性（1-5分）
- 技术熟练程度（1-5分）
- 沟通流畅度（1-5分）
- 专业性（1-5分）

客户可选择性填写文字评价。

**8.2 工程师评价客户**

工程师可在工单完成后对客户进行内部评价，维度包括：
- 配合度
- 沟通顺畅度
- 付款及时性
- 工作环境

此评价不公开展示，仅供平台内部参考。
