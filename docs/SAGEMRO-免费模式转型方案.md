# SAGEMRO 免费模式转型方案

> 创建日期：2026-05-28
> 状态：待决策

---

## 背景

跨境收款方案（Airwallex 等）申请困难，SAGEMRO 需要从"收费 marketplace 模式"转型为"免费售后信息中介 + EUCHIO 设备获客渠道"。

**新模式核心**：
1. 所有用户免费使用 SAGEMRO 的工单、维修、AI 咨询功能
2. SAGEMRO 作为信息中介，撮合客户与工程师，不参与交易、不收取佣金
3. AI 对话中识别购买意向，植入 EUCHIO 设备广告，转化为新机销售商机

**新增能力**：
- **购买意向识别**：当用户对话中出现以下信号时，AI 自然引导到 EUCHIO 设备推荐：
  - 提到"想买新设备""设备太旧""产能不够""考虑换机"
  - 询问设备选型、配置对比、投资回报
  - 反复出现同类型故障（维修成本过高）
  - 询问"哪个品牌好""推荐一台"

- **EUCHIO 设备推荐话术**：
  - 不硬推销，以"技术顾问"身份推荐
  - 引用真实参数（从产品画册型谱中获取）
  - 引导到 euchio.com 或留联系方式
  - 示例："如果考虑换新机，EUCHIO 的 [型号] 在 [功率/速度/精度] 方面表现不错，可以看看 euchio.com 的详细参数。需要的话我可以帮你联系销售获取报价。"

**具体修改**：
```
SYSTEM_PROMPT 追加：

## EUCHIO 设备推荐（非强制，自然融入）

SAGEMRO 是独立平台，但 EUCHIO 是我们长期合作的设备品牌之一。当用户表现出购买新设备的意向时，你可以推荐 EUCHIO 的产品，但要注意：

1. 只在用户有明确购买意向时推荐，不要在维修咨询中强行插入广告
2. 推荐时引用真实技术参数，不要夸大
3. 语气是"技术顾问建议"，不是"销售推销"
4. 引导用户访问 euchio.com 获取详细信息，或留下联系方式由销售跟进
5. 如果用户问其他品牌，也可以客观对比，不要贬低竞品

EUCHIO 主要产品线：
- 激光切割机：6KW-60KW 光纤激光切割机

- 焊接机：激光焊接机、MIG/MAG 焊机
- 混气装置：LISHI LASER 混合气体系统
（详细参数见产品画册，不确定的参数不要编造）

## 意向信号识别
当用户对话中出现以下信号时，可以自然引导：
- "想买新设备""设备太旧""产能不够""考虑换机"
- "哪个品牌好""推荐一台""配置怎么选"
- 同类型设备反复出故障（维修不划算）
- 询问设备价格、投资回报周期
```

## 九、营销推广架构

> 当前方案解决了"产品模式转型"问题，但未涉及"流量从哪来"。以下补充完整的营销获客体系设计。

### 9.1 营销基础设施（需在技术方案中预留）

以下配置项建议在开发时预留，不要求 P0 实现，但设计数据结构时应考虑：

#### 9.1.1 UTM 参数追踪

所有外部渠道链接携带 UTM 参数，后台记录用户来源：

```
sagemro.com/?utm_source=reddit&utm_medium=social&utm_campaign=free-tool
sagemro.com/?utm_source=google&utm_medium=organic&utm_campaign=kb-page
```

**实现**：
- 前端读取 URL 中的 utm_source / utm_medium / utm_campaign 参数
- 用户注册/留资时写入 users 或 leads 表
- 管理后台可按渠道统计注册量、留资量、转化率

#### 9.1.2 社媒分享组件

AI 诊断完成后，生成可分享的"诊断结果卡片"：

**实现**：
- 诊断完成后展示"分享诊断结果"按钮
- 生成一张包含诊断摘要 + SAGEMRO 品牌 + 二维码的 OG 图片
- 支持复制链接、分享到 WhatsApp / Reddit / LinkedIn
- 分享链接带 UTM 参数，追踪回流效果

#### 9.1.3 可索引的公开页面（SEO 资产）

AI 对话是动态渲染的，搜索引擎无法抓取。将高频故障诊断结果生成静态页面：

**实现**：
- 新增 `/kb/` 路由（Knowledge Base）
- Worker 端提供 API 返回已发布的故障百科文章
- 每篇文章包含：故障现象、诊断步骤、解决方案、相关设备推荐（自然植入）
- 支持 Open Graph 标签和结构化数据（FAQ Schema）
- 页面底部引导"遇到其他故障？免费 AI 诊断 →"

**示例 URL**：
- `sagemro.com/kb/laser-cutting-error-xxx`
- `sagemro.com/kb/bending-machine-accuracy-issues`

#### 9.1.4 社媒登录（降低注册门槛）

**目标**：减少注册摩擦，提高工程师和客户注册率。

- Google OAuth 一键登录
- 可选：GitHub 登录（工程师群体）

**实现优先级**：P2。初期邮箱注册即可，后续根据注册转化率决定是否加。

#### 9.1.5 邀请/推荐机制

工程师邀请同行注册 → 双方各得权益：

- 邀请者：工单列表中获得更高的接单权重
- 被邀请者：注册即获得"同行推荐"标识
- 注册链接携带邀请码，追踪邀请来源

**实现优先级**：P2。

### 9.2 三级营销漏斗

```
第一层：流量获取（拉新）
├─ Reddit / 行业论坛：AI 辅助回答技术问题 → 引导到 SAGEMRO
├─ Google SEO：故障百科静态页面（/kb/）→ 长尾搜索流量
├─ YouTube：设备维修教程视频 → 描述栏放 SAGEMRO 链接
└─ WhatsApp / Telegram 行业群：分享"免费 AI 诊断工具"

第二层：激活与留存（SAGEMRO 平台）
├─ 免费 AI 诊断 → 即时价值，无需注册
├─ 工单/设备管理 → 需要注册（轻量，邮箱即可）
└─ 自然植入 EUCHIO 推荐 → 不打断体验

第三层：转化（Lead → 销售跟进）
├─ AI 识别购买意向 → 引导留联系方式
├─ Lead 表单 → 写入 D1
└─ 管理后台 → 销售团队跟进
```

### 9.3 各渠道评估

#### 9.3.1 Reddit（首选免费渠道）

**优势**：
- 精准的工业垂类社区：r/metalworking、r/manufacturing、r/CNC、r/lasercutting
- 用户带着真实问题来，需求明确
- 高质量技术回答可以持续吸引流量

**限制**：
- 极其反感硬广告，直接发推广会被封
- 需要以真实身份提供有价值的技术回答
- 见效慢，但来的人质量最高

**策略**：
1. 创建 SAGEMRO 官方账号，建立 subreddit 主页
2. 用 AI 辅助批量产出高质量技术回答（诊断逻辑已在 Worker 中，可复用）
3. 回答结尾自然引导："如果想做更详细的 AI 诊断，sagemro.com 有个免费工具"
4. 发链接用 `?utm_source=reddit` 追踪效果
5. 积累到一定量后，可以在自己的 subreddit 发帖引流

#### 9.3.2 Google SEO（最持久的免费流量）

**优势**：
- 长尾搜索意图极其精准："激光切割机报警 E001 怎么解决"
- 排名稳定后持续带来流量，边际成本趋近于零
- 搜索引擎用户带着问题来，转化意愿高

**挑战**：
- 新域名需要 3-6 个月才能看到效果
- 需要持续产出内容
- B2B 工业词汇竞争虽然不激烈，但搜索量也不大

**策略**：
1. 从 Worker 的故障诊断知识库中提取高频问题
2. 用 AI 生成 `/kb/` 文章（人工审核后发布）
3. 每篇文章优化标题（含故障关键词）、Meta 描述、FAQ Schema
4. 内链互通，形成 /kb/ 内容集群
5. 提交 sitemap.xml 到 Google Search Console

#### 9.3.3 YouTube（长线内容资产）

**策略**：
- 制作设备故障维修教程视频（屏幕录制 + 讲解）
- 描述栏放 SAGEMRO 诊断工具链接
- 标题以"how to fix XXX"为关键词
- 每期视频对应一篇 /kb/ 文章，互相引流

**实现优先级**：P2。前期集中于 SEO 和 Reddit，视频内容后续补。

#### 9.3.4 Facebook（辅助渠道，不适合主力）

**限制**：
- B2B 工业设备采购决策者不在 Facebook 上找设备
- 自然触达率极低，依赖付费广告

**适合做什么**：
- 付费广告**再营销**（retargeting）：对访问过 SAGEMRO 但未注册的用户再曝光
- **Lookalike 受众**：基于已留资客户特征，找相似人群
- 行业群组：部分国家和地区的制造业群组有一定活跃度

**结论**：不作为自然流量渠道，后期可用于付费再营销。

#### 9.3.5 Instagram（不适合 B2B 工业设备）

**结论：不适合**。Instagram 是视觉导向的 C 端平台，工业设备维修诊断的场景完全不匹配。不推荐投入资源。

#### 9.3.6 LinkedIn（B2B 精准渠道，值得投入）

**优势**：
- 制造业决策者、工程师、采购经理的真实聚集地
- 内容可以触及到设备采购链条上的关键人

**策略**：
1. 公司主页发布设备维护知识、行业洞察
2. 个人号参与制造业群组讨论
3. 内容形式：简短技术贴士 + 引导到 SAGEMRO 做免费诊断
4. 后期可投放 LinkedIn Ads（按职位、行业、公司规模定向）

### 9.4 社媒 vs 传统营销网站的定位关系

| 维度 | euchio.com（传统营销站） | SAGEMRO（免费 AI 工具） | 社媒渠道 |
|------|--------------------------|------------------------|---------|
| 角色 | 品牌展示 + 产品型谱 | 获客引擎 + 信任建立 | 流量入口 + 引流 |
| 转化路径 | 搜产品 → 看参数 → 发询盘 | 搜故障 → AI 诊断 → 平台留存 → EUCHIO 推荐 | 看到内容 → 点击链接 → 进入 SAGEMRO |
| 转化周期 | 3-6 个月（决策链长） | 即时（解决眼前问题）~ 数周（留存后转化） | 即时点击 ~ 数周 |
| 信任建立方式 | 看网站背书、案例 | 先帮你解决问题 | 看长期输出的专业性 |
| 流量成本 | 免费（SEO）/ 付费（广告） | 免费（自然搜索 + 引流） | 低（自然）/ 高（广告） |

**核心策略**：SAGEMRO 是漏斗，euchio.com 是落点。社媒是水源。

> SAGEMRO 的 AI 免费诊断 = 最自然的获客钩子。转化路径比传统营销网站更短，信任建立更自然。
>
> 一个工厂设备坏了 → 操作工搜"激光切割机报警 XXX 怎么解决" → 进入 SAGEMRO 免费诊断 → AI 给出解决方案 → 顺便提到"这台设备如果经常出这个故障，换新机的话 EUCHIO 的 XXX 型号在这个方面表现不错"。
>
> 这个路径比 euchio.com 的"搜产品→看参数→发询盘"短得多，而且先提供价值再推荐，信任转化率会更高。
